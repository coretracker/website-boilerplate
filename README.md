# Website Boilerplate

### Requirements
* Nodejs
* Livereload (https://chrome.google.com/webstore/detail/livereload/jnihajbhpnppcggbcgedagnkighmdlei)

### Installation
* npm install
* npm install -g http-server

### Starting
* gulp watch in root dir
* join dist directory
* http-server
